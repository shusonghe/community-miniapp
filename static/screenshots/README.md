# 截图占位目录

请将以下页面的截图放入此目录：

## 需要的截图（建议尺寸：1080x1920 或 750x1334，压缩到 200KB 以内）

| 文件名 | 页面 | 说明 |
|--------|------|------|
| home.png | 首页 | 展示首页整体布局 |
| community.png | 社区动态 | 动态列表页面 |
| secondhand.png | 二手闲置 | 闲置商品列表 |
| group.png | 组团 | 组团活动列表 |
| notice.png | 公告 | 公告列表 |
| profile.png | 我的 | 个人中心页面 |

## 截图方法

1. 微信开发者工具 → 点击顶部「预览」→ 手机扫码打开小程序
2. 在手机上逐页截图
3. 或用微信开发者工具的截图功能：顶部菜单 → 工具 → 截图

## GIF 录制（可选，更吸引人）

- Windows: 使用 ScreenToGif 录制操作流程
- macOS: 使用 Kap 或 GIPHY Capture
- 录制时长控制在 15-30 秒，输出 GIF 格式
